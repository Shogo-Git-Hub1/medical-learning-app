# 医療系学習アプリ（Medical Learning App）

医療系学生向けの Duolingo 風学習アプリです。  
**Web アプリ**として開発し、将来 **iOS** にも展開する想定です。

## 方針

- **コンテンツ**: 資格・試験対策 ＋ 科目別（解剖学・生理学など）
- **継続施策**: ストリーク / デイリーミッション / レベル・XP / レッスンロードマップ / 間隔反復

詳細は [PLAN.md](./PLAN.md) を参照してください。

## 技術スタック

- **Next.js 15** (App Router) + **TypeScript**
- **Tailwind CSS**

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 現在の構成

- **ホーム** (`/`): ストリーク・デイリーミッション・レベル/XP のプレースホルダー
- **ロードマップ** (`/roadmap`): レッスン一覧（道のり形式）
- **レッスン** (`/lesson/[id]`): 問題表示用のプレビューページ（Phase 1 で実装）

## 今後の実装順（PLAN.md に沿って）

1. **Phase 1**: 選択式問題の表示・正誤判定、ダミーデータ
2. **Phase 2**: 5要素の実装（ストリーク・デイリーミッション・レベル/XP・ロードマップ状態・間隔反復）
3. **Phase 3**: 科目・試験タグ、コンテンツ拡張
4. **将来**: PWA または Capacitor で iOS 配布
