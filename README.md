# MediSpark

> 医療系学生が **1日5分で国家試験レベルの知識を反復できる**クイズアプリ

医師・看護師・薬剤師・歯科医師などの国家試験を受験する学生向けに、
Duolingo 風のゲーミフィケーションで「毎日続けたくなる」学習体験を提供します。
**Web アプリ（PWA）**として開発し、将来 **iOS** にも展開する想定です。

## 方針

- **コアコンセプト**: 1日5分・短いセッション・繰り返しによる記憶定着（間隔反復）
- **コンテンツ**: 国家試験対策 + 科目別（解剖学・生理学・薬理学・病理学など）
- **継続施策**: ストリーク / デイリーミッション / レベル・XP / 科目別ロードマップ / 間隔反復
- **UX方針**: 直感的・進捗がひと目で分かる UI（プログレスバーなど）

詳細は [PLAN.md](./PLAN.md) を参照してください。

## 技術スタック

- **Next.js 15** (App Router) + **TypeScript**
- **Tailwind CSS**（ニューモーフィズム × パステルデザイン）

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 現在の構成

- **ホーム** (`/`): ストリーク・デイリーミッション・レベル/XP（進捗連動）
- **科目** (`/subjects`): 科目カード一覧（科目ごとにロードマップへ遷移）
- **科目ロードマップ** (`/subjects/[subject]`): 科目内レッスン一覧（完了で次アンロック）
- **ブラウズ** (`/browse`): 科目・試験タグでフィルタしてレッスン一覧を表示
- **レッスン** (`/lesson/[id]`): 選択式問題＋復習問題の出題（各問題に「この問題を報告」リンクあり）
- **プロフィール** (`/profile`): ユーザー設定・お問い合わせ（準備中）
- **お問い合わせ** (`/contact`): 問い合わせフォーム・問題の誤り・誤字脱字の報告（mailto で送信）

## PWA（ホーム画面に追加）

- HTTPS で配布すると、スマホ・PC で「ホームに追加」「アプリとしてインストール」が可能です。
- マニフェスト・テーマ色・アイコンは `app/manifest.ts` と `layout.tsx` の viewport で設定済みです。

## iOS ビルド（Capacitor）

- 静的エクスポート（`output: "export"`）でビルドし、Capacitor で iOS アプリとしてビルドできます。
- **macOS が必要です**（Xcode でビルドするため）。

```bash
npm run build    # 静的ファイルを out/ に出力
npm run cap:sync # out/ を ios プロジェクトにコピー
npm run ios      # ビルド → コピー → Xcode を開く（一括）
```

Xcode が開いたら、シミュレータまたは実機を選んで実行してください。
※ 本番配布時は `ios/App/App.xcworkspace` で署名・アーカイブを行います。

- **補足**: `npm run start` は静的エクスポート後の `out/` をローカルで配信します（開発時は `npm run dev` を使用）。

## トラブルシューティング

- **`Cannot read properties of undefined (reading 'call')`** が出る場合: Service Worker のキャッシュやデプロイ後のバージョンずれが原因のことが多い。原因・対策・開発時の対処は [docs/WEBPACK_CALL_ERROR_ROOT_CAUSE.md](./docs/WEBPACK_CALL_ERROR_ROOT_CAUSE.md) にまとめてある。

## テスト

- **単体テスト（Vitest）**: 進捗・復習・レッスンローダーのロジックをテストします。
  ```bash
  npm run test        # 1回実行
  npm run test:watch  # ウォッチモード
  ```
- **E2E（Playwright）**: ホーム → 科目 → ロードマップ → レッスン → 1問解答、復習ページ・ブラウズの表示を検証します。
  ```bash
  npm run dev         # 別ターミナルでサーバーを起動
  npm run test:e2e    # E2E 実行（起動中の localhost:3000 を使用）
  ```
