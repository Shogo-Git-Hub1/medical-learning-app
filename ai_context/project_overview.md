# プロジェクト概要（AI向けコンテキスト）

## このプロジェクトとは

**MediSpark**（メディスパ）— 医療系学生が1日5分で国家試験レベルの知識を反復できるクイズアプリ。
Web・PWA・iOS で動作する。
Duolingo 風のゲーミフィケーション（ストリーク・XP・レベル・デイリーミッション・間隔反復）で継続学習を促す設計。

## 対象ユーザー

医師・看護師・薬剤師・歯科医師などの国家試験を受験する学生。

## 現在の機能（実装済み）

- 選択式クイズ（4択）・正誤判定・解説表示
- 科目別ロードマップ（解剖学・生理学・薬理学・病理学）とアンロック制御
- ブラウズ（科目・試験タグでのフィルタ）
- ストリーク、デイリーミッション（5問/日）、レベル・XP（正解+10XP）
- 間隔反復（SM-2準拠の簡易アルゴリズム）
- PWA対応（オフライン可、ホーム追加可）
- Capacitorを使ったiOSビルド
- お問い合わせ・問題報告（mailtoリンク方式）

## 現在のデータ規模

- 解剖学: 2レッスン / 6問
- 生理学: 1レッスン / 3問
- 薬理学: 1レッスン / 2問
- 病理学: 1レッスン / 2問
- 合計: 5レッスン / 13問（継続的に拡充中）

## 技術的制約

- **静的エクスポート必須**: Capacitorとの連携のため `output: "export"` を使用。サーバーサイドのAPIルートは使用不可
- **認証なし**: 現在は localStorage のみ。進捗はデバイスローカル
- **バックエンドなし**: 問題データはコードに直接記述

## 今後の方向性

- コンテンツ拡充（問題・レッスンの継続追加）
- プロフィール機能（ユーザー設定・学習統計）
- 将来的にSupabaseへ移行してクラウド進捗同期を実現
- App Storeへの申請（現時点では未定）

## 重要な設計決定

1. **App Router維持**: Next.js 15のApp Routerを使用。`src/pages/` は存在しない
2. **サービス層分離**: `src/services/` にデータアクセスを集約し、DB移行時の変更範囲を最小化
3. **コンテンツはコードで管理**: CMSは使わず TypeScript ファイルで問題を管理（型安全性・バージョン管理のため）
4. **ロードマップは科目ごと**: 複数科目をまたぐグローバルロードマップは持たない。各科目ページ（`/subjects/[subject]`）から科目内ロードマップへ遷移する
