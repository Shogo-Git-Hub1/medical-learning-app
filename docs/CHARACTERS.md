# キャラクター設定（医療系学習アプリ ナビゲーター）

実装・イラスト作成時の**正**として参照するドキュメントです。

---

## 1. 全体コンセプト

医療系学生（医学生、看護学生、薬学生など）の膨大な暗記学習をサポートし、モチベーションを維持するためのナビゲーターキャラクター。

- **テーマ**: 「一緒に成長する」「親しみやすさ」「クスッと笑える可愛さ」
- **構成**: 3人組のチーム

---

## 2. プロフィール

### ① スカルン（ガイコツ）

| 項目 | 内容 |
|------|------|
| **役割** | 癒やし・共感担当（ユーザーと同じ学習者目線） |
| **ビジュアル** | シンプルでぽてっとした2頭身フォルム。骨なのに**ピンクのほっぺ**があり、表情豊かで可愛い。 |
| **性格・アクション** | 自分の骨の名前をすぐ忘れる、少し抜けた愛されキャラ。間違えるとカタカタ震えて一緒に落ち込み、正解するとバラバラになって喜ぶ。 |

### ② レジ先生（ドクター）

| 項目 | 内容 |
|------|------|
| **役割** | 論理・解説担当（知識の導き手） |
| **ビジュアル** | サイズが合っていない**ぶかぶかの白衣**を着ており、裾を引きずって歩いている（研修医＝レジデントの未熟さを表現）。 |
| **性格・アクション** | クールで教え上手だが、まだ半人前でドジな一面もある。スカルンがボケた時のツッコミ役。難しい知識を分かりやすく論理的に解説する。 |

### ③ シリンちゃん（ナース）

| 項目 | 内容 |
|------|------|
| **役割** | 鼓舞・スケジュール担当（学習のペースメーカー） |
| **ビジュアル** | 自分の身長ほどある**特大の注射器ペン**を抱えている。 |
| **性格・アクション** | 明るくてパワフル、学生を元気付けるムードメーカー。「ここはテストに出るよ！」と大きな注射器ペンで重要事項をメモしたり、ログインボーナスを配ったりする。 |

---

## 3. 関係性（チームワーク）

- **スカルンの出自**: レジ先生が学生時代に使っていた（または今も使っている）人体模型に魂が宿った存在。
- **掛け合いの例**:
  - **レジ先生**: 「おいスカルン、自分の大腿骨の名前くらい覚えろよ」
  - **スカルン**: 「えへへ、忘れちゃった（テヘペロ）」
  - **シリンちゃん**: 「もう、レジ先生も厳しくしないの！スカルン、一緒に頑張ってメモるよ！」

---

## 4. 成長（進化）ルール

学習の進捗（レベル・正答率）に合わせて、キャラクターの見た目が進化する。

### スカルン

| レベル | ステージ | 見た目 |
|--------|----------|--------|
| Lv.1〜4 | bone | 骨のみ（基本） |
| Lv.5〜9 | muscle | 筋肉がつく |
| Lv.10〜19 | nerve | 神経が通る |
| Lv.20〜 | full | 立派なフルカラー人体模型に近づく |

### レジ先生・シリンちゃん

| レベル | ステージ | 見た目 |
|--------|----------|--------|
| Lv.1〜9 | trainee | ぶかぶか白衣・研修医風 |
| Lv.10〜 | full | 白衣・スクラブが立派にランクアップし、一人前の医療従事者の姿に |

※ 試験が近いなどの「特別条件」は将来拡張で追加可能（現状はレベルのみで判定）。

## 5. 画面別の使い方

| 画面 | 主に使うキャラ | 用途 |
|------|----------------|------|
| **ホーム** | 3人 or スカルン＋シリン | 挨拶・「今日も一緒に頑張ろう」、デイリーミッション/ストリークの横にナビゲーター。進捗に応じた見た目（進化ステージ）を表示。 |
| **クイズ（問題中）** | スカルン（＋必要ならレジ） | 正解時: 喜びの反応（バラバラになって喜ぶ）。不正解時: カタカタ震えて一緒に落ち込む。解説時: レジの一言（「正解は〇〇です」など）。 |
| **クイズ（フィードバック）** | スカルン / レジ | 正解メッセージをスカルン、解説をレジの台詞として表示。 |
| **レッスン完了** | 3人 or シリン中心 | 「おつかれさま！」「ここテストに出るよ！」（シリン）。正解数に応じてスカルンが喜ぶなど。 |
| **ロードマップ** | シリン or スカルン | 「次はここをやろう」などペースメーカー的メッセージ。 |
| **ブラウズ** | 任意 | 「シリンちゃんのオススメ」など軽いラベル・台詞。 |
| **お問い合わせ/問題報告** | 任意 | 送信後のサンクスで「報告ありがとう」など1キャラの台詞。 |

---

## 6. コード・データとの対応

- **型・ステージ定義**: `src/types/characters.ts`
- **プロフィール・台詞・レベル→ステージ**: `src/data/characters.ts`
- **表示コンポーネント**: `CharacterAvatar`, `CharacterLine`（`src/components/` 配下）
- **3人並び画像**: `public/characters/navigators.png`（左からスカルン・レジ先生・シリンちゃん）

台詞の文言を変える場合は `src/data/characters.ts` を編集すると、全画面の台詞が一括で変わります。
